Ori TTS API Documentation
==========================

Welcome to the Ori TTS API documentation. This API provides text-to-speech synthesis with support for multiple voices, languages, and audio formats.

Features
--------

* **Real-time Streaming**: Stream audio via HTTP or WebSocket
* **OpenAI Compatible**: HTTP endpoint follows OpenAI TTS API format
* **Voice Cloning**: Create custom voices from audio samples
* **Multiple Formats**: Support for MP3, PCM, and Î¼-law encoding
* **Multiple Languages**: Support for English (US) and Hindi
